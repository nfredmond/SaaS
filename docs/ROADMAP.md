# Roadmap (Solo, ASAP)

## Phase 0: Spec + Setup (1 week)
- Define MVP scope and data model.
- Scaffold Next.js app + Supabase project.
- Set up local DuckDB cache.

## Phase 1: Analytics-First MVP (4-6 weeks)
- Ingest: ACS, LODES, FARS + local CSV/GeoJSON.
- Map UI with layers + metric cards.
- Accessibility + HIN + equity metrics.
- Auto report PDF.

## Phase 2: Engagement Loop (3-4 weeks)
- Public input page (survey + map pins).
- Sentiment summary + representation dashboard.
- Link engagement data to analytics.

## Phase 3: Scenario Planning (6-10 weeks)
- Before/after corridor scenario comparisons.
- Multi-metric dashboards and export.

## Phase 4: Compliance Modules (later)
- FHWA report pack templates.
- NEPA/CEQA tracking workflows.
